The WoW Character Journal Companion App turns your gameplay into a living story. Working alongside the in-game addon, it captures detailed session data—your travels, battles, deaths, quests, and key events—and organizes it into a structured timeline.

After your play session, simply run the companion app to generate a rich, narrative-ready prompt. This prompt is designed to be pasted into the LLM of your choice (such as ChatGPT or others), which then transforms your session into an immersive, in-character journal entry.

You can define your character’s identity, personality, and tone—whether that’s a cynical Dark Ranger, a noble paladin, or a wandering druid—and the generated prompt ensures the final story reflects that voice consistently.

Rather than producing generic logs, the app enriches your session data with context and meaning, helping the LLM interpret events naturally—turning movement into journeys, combat into encounters, and quests into story arcs. The result is a clean, expressive narrative that feels like a true chronicle of your character’s journey.

Designed to be lightweight and flexible, the companion app doesn’t lock you into a specific AI tool—instead, it empowers you to use any LLM you prefer while giving you the best possible input to create high-quality, personalized stories.

Requires the WoW Character Journal Addon to function

Instructions:
Download EXE and place it in a folder of your choice.
Run it immediately after a play session.
It will ask you for your WoW folder.
Then you can select a session to extract.
You can then add tags, these can be things like personality traits or general traits. I add Dark Ranger to mine.
It will ask you POV. Select character if you want an in character journal and narrator if you want more of a recap. Tags still work for narrator.
If you choose narrator you will have to choose narrator type.
When all this is done it will extract and generate your session and prompt.
These will be in an output folder in the same folder as the exe.
Copy and paste the complete prompt file (or just upload it) to an llm of your choosing, I usually go with ChatGPT.
